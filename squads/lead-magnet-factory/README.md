# Lead Magnet Factory Squad

Squad especializada em criar lead magnets de alta conversão para topo de funil: diagnósticos/assessments, templates/frameworks e calculadoras/simuladores.

## Propósito

Criar iscas que:
- Capturam leads qualificados (ICP KOSMOS)
- Demonstram expertise antes de vender
- Geram dados para personalização
- Iniciam relacionamento de confiança

## Tipos de Lead Magnet

### 1. Diagnósticos/Assessments
Ferramentas interativas que geram relatório personalizado.
- Ex: KOSMOS Score (diagnóstico de ativos digitais)
- Perguntas estruturadas → Scoring → Relatório → CTA

### 2. Templates/Frameworks
Materiais prontos para uso imediato.
- Ex: Template de Oferta High Ticket
- Download → Aplicação → Valor percebido → Desejo de mais

### 3. Calculadoras/Simuladores
Ferramentas que mostram potencial de resultado.
- Ex: Calculadora de Potencial de Faturamento
- Input de dados → Cálculo → Visualização → CTA

## Fluxo de Criação

```
1. ESTRATÉGIA (lead-magnet-strategist)
   └── Definir tipo, posicionamento, dor endereçada

2. ARQUITETURA (diagnostic-architect / template-builder / calculator-designer)
   └── Estruturar a mecânica do lead magnet

3. CAPTURA (landing-copywriter)
   └── Copy para landing page

4. NUTRIÇÃO (nurture-architect)
   └── Sequência pós-captura

5. VALIDAÇÃO (icp-analyst)
   └── Verificar alinhamento com ICP
```

## Métricas de Sucesso

Para cada lead magnet criado:
- Taxa de conversão da landing (target: 30%+)
- Taxa de abertura do nurture (target: 40%+)
- Taxa de agendamento de call (target: 5%+)
- NPS do lead magnet (target: 50+)
