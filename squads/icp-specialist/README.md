# ICP Specialist Squad

Squad especializada em aplicar o conhecimento arqueológico profundo do ICP KOSMOS (creators digitais) para criar conteúdo que ressoa e outbound que converte.

## Propósito

Transformar o documento de ICP de 50+ páginas em ações concretas:
- Mensagens de outbound personalizadas por perfil
- Conteúdo que atinge as 3 portas emocionais
- Respostas preparadas para as 12 objeções
- Timing preciso de abordagem

## Agentes

| Agente | Função |
|--------|--------|
| `icp-analyst` | Analisa situações através das 7 camadas de dor e 5 perfis |
| `outbound-strategist` | Cria sequências de DM/email usando linguagem exata do ICP |
| `content-strategist` | Planeja conteúdo para as 3 portas emocionais |
| `objection-handler` | Prepara respostas para as 12 objeções mapeadas |
| `timing-analyst` | Identifica janelas de receptividade e micro-momentos |

## Tasks

| Task | Descrição |
|------|-----------|
| `analyze-icp-fit` | Verifica alinhamento de copy/conteúdo com ICP |
| `create-outbound-sequence` | Cria cadência de outbound (DM/email) |
| `create-content-plan` | Plano de conteúdo baseado nas dores |
| `map-buyer-journey` | Mapeia jornada de decisão |
| `generate-objection-responses` | Gera respostas para objeções |
| `identify-timing-windows` | Identifica melhor momento de abordagem |

## Knowledge Base

A squad usa como base de conhecimento:
- `data/icp-creator-kosmos.md` - Documento ICP completo
- `data/perfis-icp.yaml` - 5 perfis estruturados

## O ICP KOSMOS

### As 7 Camadas de Dor
1. Frustração Financeira Silenciosa
2. Síndrome do Impostor Invertida
3. Dependência do Algoritmo
4. Solidão Estratégica
5. Medo da Irrelevância
6. Culpa com a Base
7. Ceticismo com o Mercado

### Os 5 Perfis
1. **Professor Preso** - Sabe ensinar, não sabe vender
2. **Lançador Cansado** - Exausto do ciclo de lançamentos
3. **Community Builder Frustrado** - Tentou comunidade e falhou
4. **Expert Técnico** - Muito bom, não se vê como creator
5. **Creator-Empresário Travado** - Pensa como empresário, travado no modelo

### As 3 Portas Emocionais
1. **Alívio** - "Finalmente alguém que entende"
2. **Inveja Estratégica** - "Ele conseguiu, eu também consigo"
3. **Esgotamento** - "Eu não aguento mais"

## Uso

```bash
# Invocar agente
@icp-analyst

# Executar task
*analyze-icp-fit
*create-outbound-sequence
```
