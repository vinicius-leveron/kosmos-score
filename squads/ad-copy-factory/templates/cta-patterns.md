# Template: Padroes de CTA

Referencia rapida para criar CTAs de alta conversao.

---

## Formulas de CTA

### 1. Acao + Beneficio

**Estrutura:**
```
"[Verbo] + [O que ganha]"
```

**Exemplos:**
- "Descobrir meu potencial oculto"
- "Calcular minha receita"
- "Ver meu diagnostico"
- "Encontrar meu lucro escondido"
- "Revelar meu score"

**Quando usar:** CTAs de botao, chamadas principais

---

### 2. Acao + Especificidade

**Estrutura:**
```
"[Verbo] + [Elemento especifico do LM]"
```

**Exemplos:**
- "Descobrir meu KOSMOS Score"
- "Fazer o diagnostico gratuito"
- "Calcular meu lucro oculto"
- "Acessar a calculadora"
- "Pegar meu template"

**Quando usar:** Quando o LM tem nome proprio

---

### 3. Acao + Tempo

**Estrutura:**
```
"[Verbo] + em [tempo]"
```

**Exemplos:**
- "Descobrir em 3 minutos"
- "Ver resultado agora"
- "Comecar diagnostico"
- "Calcular em 2 minutos"
- "Saber em 60 segundos"

**Quando usar:** Reduzir atrito, enfatizar rapidez

---

### 4. Imperativo Direto

**Estrutura:**
```
"[Verbo imperativo] + [objeto]"
```

**Exemplos:**
- "Faca o diagnostico"
- "Calcule seu potencial"
- "Descubra seu score"
- "Baixe o template"
- "Acesse a calculadora"

**Quando usar:** Tom mais urgente/direto

## Textos de Suporte (Redutores de Atrito)

Use abaixo ou ao lado do CTA principal:

### Sobre Compromisso
- "Gratuito, sem compromisso"
- "100% gratuito"
- "Sem precisar de cartao"
- "Sem cadastro complexo"

### Sobre Tempo
- "Leva menos de 3 minutos"
- "Resultado instantaneo"
- "Em 60 segundos"
- "Rapido e simples"

### Sobre Formato
- "100% online"
- "No seu celular"
- "Direto no navegador"
- "Sem baixar nada"

### Sobre Privacidade
- "Seus dados protegidos"
- "Resultado so para voce"
- "Privado e seguro"

## Combinacoes Recomendadas

### Para Botao + Suporte

**Diagnostico:**
```
Botao: "Fazer Diagnostico Gratis"
Suporte: "Leva menos de 3 minutos"
```

**Calculadora:**
```
Botao: "Calcular Meu Potencial"
Suporte: "Resultado instantaneo, sem compromisso"
```

**Template:**
```
Botao: "Baixar Template Gratis"
Suporte: "100% editavel, sem cadastro"
```

### Para Texto do Anuncio

**Opcao 1 - Beneficio:**
```
"Faca o diagnostico gratuito e descubra seu potencial oculto."
```

**Opcao 2 - Tempo:**
```
"Em 3 minutos, descubra quanto voce esta deixando na mesa."
```

**Opcao 3 - Acao direta:**
```
"Clique no link e veja seu resultado agora."
```

## Checklist de CTA

- [ ] Tem verbo de acao
- [ ] Inclui beneficio ou especificidade
- [ ] Maximo 5 palavras
- [ ] Reduz atrito percebido
- [ ] Especifico ao lead magnet
- [ ] Nao usa linguagem que repele
- [ ] Funciona isolado (faz sentido sozinho)
- [ ] Tem texto de suporte se necessario

---

## Matriz de Selecao

| Situacao | Formula Recomendada |
|----------|---------------------|
| LM com nome forte | Acao + Especificidade |
| LM generico | Acao + Beneficio |
| Publico frio | Acao + Tempo |
| Publico quente | Imperativo Direto |
| Muita objecao | Adicionar texto de suporte |
