# Template: Anuncio Estatico

Use este template para estruturar anuncios estaticos para Meta Ads.

---

## Formato Padrao

```markdown
# Anuncio: [Nome do Lead Magnet]

## Informacoes
- **Lead Magnet:** [nome]
- **Tipo:** [diagnostico/template/calculadora]
- **Perfil alvo:** [tipo do ICP]
- **Tom:** [empoderador/provocativo/urgente]

---

## Copy do Anuncio

### Hook (1a linha)
> [Texto que para o scroll - max 15 palavras]

### Problema (2-3 linhas)
[Identificar a dor]
[Mostrar que entende]
[Criar conexao]

### Agitacao (1-2 linhas)
[Amplificar a dor]
[Mostrar consequencias]

### Solucao (2-3 linhas)
[Apresentar o lead magnet]
[O que descobre/ganha]
[Beneficio tangivel]

### CTA (1 linha)
> [Chamada clara para acao]

---

## Texto Completo

---

[HOOK]

[PROBLEMA - paragrafo 1]

[PROBLEMA - paragrafo 2]

[AGITACAO]

[SOLUCAO - paragrafo 1]

[SOLUCAO - paragrafo 2]

[CTA]

---

## Elementos do Anuncio

### Texto do Botao
> [Ex: "Fazer Diagnostico Gratis"]

### Headline do Link
> [Ex: "Descubra seu potencial oculto"]

### Descricao do Link
> [Ex: "Diagnostico gratuito em 3 minutos"]

---

## Metricas

| Metrica | Valor | Ideal |
|---------|-------|-------|
| Caracteres | [X] | 125-250 |
| Linhas | [X] | 8-12 |
| Tempo leitura | [X]s | 15-30s |

---

## Direcao Visual (para Designer)

### Tom Visual
[Ex: Profissional mas acessivel]

### Elemento Principal
[Ex: Pessoa pensando / Grafico / Smartphone]

### Cor Dominante
[Ex: Azul KOSMOS / Verde confianca]

### Texto na Imagem (se houver)
[Ex: "Quanto voce esta deixando na mesa?"]

### Formato
- [ ] Feed quadrado (1080x1080)
- [ ] Feed vertical (1080x1350)
- [ ] Stories (1080x1920)
- [ ] Reels (1080x1920)

---

## Checklist Pre-Publicacao

### Copy
- [ ] Hook para o scroll em 1 segundo
- [ ] Problema claro e especifico
- [ ] Agitacao sem ser manipulativo
- [ ] Solucao conectada ao problema
- [ ] CTA com acao + beneficio
- [ ] Linguagem do ICP
- [ ] Sem promessas irrealistas
- [ ] Flui naturalmente quando lido

### Tecnico
- [ ] URL do lead magnet correta
- [ ] Pixel configurado
- [ ] UTMs definidos
- [ ] Botao correto selecionado

### Visual
- [ ] Imagem aprovada
- [ ] Texto < 20% da imagem
- [ ] Contraste adequado
- [ ] Mobile-friendly
```
